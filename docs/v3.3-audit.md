# v3.3 macOS assumption audit

Source of truth: `docs/v3.3-roadmap.md`.

## Ubuntu 24.04 test baseline

Goal: run `make test` in a fresh Ubuntu 24.04 environment before changing code.

Current local blocker:

- `docker info` can reach the Docker client, but the default sandbox cannot access the Docker socket:
  `permission denied while trying to connect to the docker API at unix:///Users/chris/.docker/run/docker.sock`.
- With elevated Docker access, `docker run --rm ... ubuntu:24.04 ... make test` could not reach the test phase. Docker reported
  `Unable to find image 'ubuntu:24.04' locally`, then remained silent during image acquisition until killed.
- A direct `docker pull ubuntu:24.04` also remained silent until killed.
- The local preexisting Linux devcontainer image is Debian 12, not Ubuntu 24.04. It is not a substitute baseline; it fails before tests with:
  `bash: line 1: bundle: command not found`.

Follow-up proof must come from the Ubuntu CI job added in Section 3.
